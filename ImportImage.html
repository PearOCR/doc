# 加载图像

PearOCR提供了多种加载图像的方法，尽可能地简化使用过程

支持同时选取/拖拽多张图片进行识别

# 通过对话框选取本地图像

点击[本地图片]按钮，从弹出的对话框中选择要识别的图片

# 拖动文件进行识别

除了通过打开选择对话框，还可以直接把文件拖到识别框内进行识别

# 剪贴板加载

PearOCR可以直接读取剪贴板的图像，配合截图工具（如微信，QQ），截图完直接识别，无需另存为本地文件

除了点击按钮，也可以通过Ctrl+V快捷键加载剪贴板中的图像
